This is a simple HTTP server built from scratch in Rust

Features

  • Serves static HTML, CSS, and JS
  • Handles HTTP GET requests
  • logs requests paths to the terminal

How to run

cargo run

Then open your browser at http://127.0.0.1:7878

The url will also appear in your terminal

Notes

This server is intended for learning purposes and local development. Do not expose it to the open web without adding proper security.
